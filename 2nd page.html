What are the Different Types of Sources?

Primary Sources

  • These are sources that come from the origin of information itself, which has not been interpreted by someone.
  • Examples of these are:
    diaries, speeches, interviews, manuscripts, novels, poetry, music, pottery, and many more.

Secondary Sources

  • These are sources that have been interpreted and described, where this source did not experience or participate first-hand in what happened.
  • Examples of these are:
    bibiliography, commentaries, essays, magazine articles, newspaper articles, reprints of art work, and many more.

Tertiary Sources

  • These sources present summarized versions of materials and/or condensed ones.
  • Examples of these are:
    almanacs, dictionaries, encyclopedias, and many more.

Importance of Knowing the Types of Sources

  • Different needs for information calls for a different type of source; for example, if an autobiography of Jose Rizal is to be used, but a biography was used, there may be inaccurate data used in the biography compared to the original manuscript of his autobiography says.
  • Using the right type of source to present arguments and research is needed since different sources have different authenticities and using the right type of source would make the argument stronger and harder to dispute.
  • Knowing the type of source is also essential in citing references as different types of sources call for different types of references so that plagiarism would not occur.

Importance of Using Credible Sources

It is very important to use credible and reliable sources, particularly in a professional setting, because it would result in people's work leading to misinformation especially without any proof supporting claims or data or using unapproved sources.
